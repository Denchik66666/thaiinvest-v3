# Структура страниц дашборда ThaiInvest (снимок на момент генерации)

Кратко: что на странице, какие блоки и какие данные обычно показываются. Роли: **SUPER_ADMIN / OWNER** видят сеть и управление; **INVESTOR** — свой кабинет и финансы.

---

## `/dashboard` — главная

- **Шапка:** аватар и имя (→ профиль), уведомления, переключатель темы; при скролле — тень.
- **Блок недели:** заголовок по роли (инвестор: неделя; владелец: число активных; супер-админ: число инвесторов и дата выплаты) + компактная шкала Пн–Вс и прогресс недели.
- **Метрики:** плитки «стекло» — для инвестора: баланс (акцент), начислено, выплачено; для владельца/супер-админа: тело в сети и др.
- **Действия (условно):** карточка «К выплате → Вывести» при наличии due; для OWNER/SUPER_ADMIN — «N заявок ожидают» → отчёты.
- **Список позиций:** карточки инвесторов (имя, статус, тело / начислено / к выплате) или пустое сообщение; у SUPER_ADMIN — фильтр Все / Общая / Приватная.
- **Низ:** `MobileBottomNav` (Главная, Инвесторы, Управление, Отчёты; у инвестора «Управление» ведёт в профиль).

**Данные:** из `GET /api/investors` (сеть по роли), агрегаты по «моим» или всем позициям.

---

## `/dashboard/investors` — список позиций

- **Шапка** как на дашборде.
- **Заголовок:** «Список позиций», подпись недели при необходимости.
- **Таблица / карточки:** фильтры сети, поиск, строки с инвесторами (имя, владелец, суммы, статус, действия).
- **Низ:** нижняя навигация.

**Данные:** список инвесторов с API (поля body, accrued, paid, due, status, owner…).

---

## `/dashboard/investors/[id]` — карточка инвестора

- **Шапка** + назад.
- **Карточка позиции:** `InvestorCard` — имя, статус, тело, ставка, начислено, выплачено, к выплате; формы запроса выплаты (по правам).
- **Для инвестора:** быстрые ссылки (финансы, отчёты по инвестору).
- **Для админа:** блок административных действий (сброс доступа, удаление), выдача учётных данных.
- **«История и расчёт»:** сводка недель + таблица последних недель (ledger).
- **Низ:** навигация (у инвестора может подсвечиваться «финансы» на детальной — зависит от страницы).

**Данные:** `GET` деталей инвестора + weekly ledger.

---

## `/dashboard/manage` — управление (OWNER / SUPER_ADMIN)

- **Шапка.**
- **SUPER_ADMIN:** предупреждение о готовности системы при необходимости; блок «Система»; компактный обзор сети.
- **Ставка бизнеса:** `BusinessRateControlCenter` (OWNER и SUPER_ADMIN).
- **Быстрый доступ:** создать инвестора, реестр, финансы, профиль; у OWNER — строка счётчиков очередей.
- **Создать инвестора**, модалка создания.
- **Низ:** навигация.

**Данные:** готовность (SUPER_ADMIN), текущая ставка, история ставок (для UI/модалки), инвесторы/запросы (OWNER).

---

## `/dashboard/profile` — профиль

- **Шапка.**
- **Вкладки:** данные аккаунта / настройки / безопасность.
- **Данные пользователя:** username, аватар, смена пароля.
- **Уведомления:** звук, вибрация, режим опроса.
- **SUPER_ADMIN:** секция сброса БД (опасная зона).
- **Низ:** навигация.

**Данные:** текущий пользователь (`/api/auth/me`), локальные prefs уведомлений.

---

## `/dashboard/finance` — финансы (**только INVESTOR**)

- При другой роли — **редирект** на `/dashboard/manage`.
- **Шапка** + «Назад».
- **Обзор позиций** инвестора: суммы по телу / начислено / к выплате, список позиций.
- **Низ:** навигация (у инвестора 4 пункта, «Управление» → профиль).

**Данные:** `GET /api/investors` в режиме инвестора.

---

## `/dashboard/chat` — чат

- **Шапка.**
- **Список партнёров** / переписка или заглушка «Чат временно отключён» в зависимости от API/флагов.
- Поле ввода сообщения при активном чате.
- **Низ:** навигация.

**Данные:** контекст чата, партнёры, сообщения (если включено).

---

## `/dashboard/reports` — отчёты

- **Шапка.**
- **Заголовок** «Отчёты», фильтры по инвестору / периоду (по реализации страницы).
- **Секции:** заявки на выплаты, пополнения тела, аудит / история (collapsible).
- **Низ:** навигация.

**Данные:** обогащённые платежи, топ-апы, аудит с API отчётов.
