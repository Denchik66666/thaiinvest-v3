Важно сначала прочитать (чтобы не было путаницы)
============================================

• Никакого «отдельного сервера агента» нет. Сайт лежит там, куда ТЫ его
  подключил (обычно Vercel). База — в ТВОЁМ проекте Supabase.
  Агент работает только внутри Cursor на твоём компьютере и не может
  сам зайти в твой Supabase без того, что ты один раз положишь адреса
  в файл .env на диске.

• Пароли и полные строки подключения в этот чат НЕ пишем — только в .env
  у себя на компьютере.


Шаг 1 — Supabase (браузер)
==========================

В новом интерфейсе Supabase строки подключения чаще всего так:

  A) Открой свой проект на https://supabase.com/dashboard

  B) Вверху страницы проекта найди кнопку **Connect** (или «Подключиться»)
     и нажми её.

  C) Откроется окно с вариантами подключения. Тебе нужно скопировать ДВА
     разных URI (кнопки Copy рядом со строкой):

     1) **Direct connection** (прямое подключение) — хост обычно вида
        db.XXXX.supabase.co и порт **5432**.
        Эту строку вставишь в файл .env как DIRECT_URL= "..."

     2) Для приложения (Vercel / Next) — обычно **Session** или
        **Transaction** pooler (Supavisor) — порт может быть **5432**
        (session) или **6543** (transaction), хост часто
        aws-0-....pooler.supabase.com
        Эту строку вставишь как DATABASE_URL= "..."

Если кнопки **Connect** не видно — открой слева шестерёнку **Project Settings**,
потом слева **Database** и пролистай страницу вниз: блоки вроде
**Connection string** / **Database password** / **URI**.

Пароль к базе, если просят — задаётся или показывается в тех же
настройках Database (кнопка «Reset password» только если забыл —
после сброса придётся обновить строки в Vercel и в .env).


Шаг 2 — Файл .env в Cursor (на твоём ПК)
========================================

1. В Cursor открой папку проекта ThaiInvest.

2. Открой файл **.env** (если нет — скопируй **.env.example** в новый файл
   с именем **.env**).

3. Вставь ТОЛЬКО у себя в файле (пример, кавычки оставь как в примере):

   DATABASE_URL="вставь сюда строку из Connect: Session или Transaction"
   DIRECT_URL="вставь сюда строку из Connect: Direct (порт 5432 к db....)"
   JWT_SECRET="любая длинная случайная строка"
   NEXT_PUBLIC_APP_URL="https://твой-сайт.vercel.app"

4. Сохрани файл (Ctrl+S).


Шаг 3 — Напиши агенту
=====================

В чат напиши одно слово: **готово** (без копирования сюда содержимого .env).


Если совсем не находишь Connect
================================

Напиши агенту: **«нет кнопки Connect»** и опиши одной фразой, что видишь
слева в меню (например: есть только Home, Table Editor, SQL, Storage).
Агент подскажет по твоему скриншоту следующий клик — скрин можно без
паролей на экране.


Если «ничего не вижу» — разберись, ГДЕ именно пусто
===================================================

A) Не видишь кнопку Connect в Supabase
--------------------------------------
• Зайди на https://supabase.com/dashboard — должен быть список проектов.
  Нажми НАЗВАНИЕ проекта (не «New project»), откроется главная проекта.
• Вверху справа часто зелёная кнопка **Connect** / «Подключить».
  Если её нет — язык интерфейса English (внизу слева профиль → Language).
• Запасной путь БЕЗ Connect:
  слева внизу **шестерёнка** (Settings / Настройки проекта) → слева пункт
  **Database** → прокрути страницу вниз до блоков с URI / Connection string.

B) Не видишь файл .env в Cursor
--------------------------------
• Слева панель **Explorer** (иконка двух листочков). Файл **.env** может
  быть в самом низу списка (точка в начале имени).
• Если пусто: **File → Open Folder** и выбери папку «THAIINVEST v3.0».
• Если .env нет: в проводнике Windows зайди в папку проекта — есть ли
  файл **.env.example**? Скопируй его и переименуй копию в **.env**,
  открой в Cursor.
• Быстрый открыть файл: **Ctrl+P**, набери `.env`, Enter.

C) Файл .env есть, но не понятно, КУДА вставлять строки
--------------------------------------------------------
• Каждая настройка — ОТДЕЛЬНАЯ строка, с новой строки, например:
  DIRECT_URL="..."
  DATABASE_URL="..."
  (без пробела вокруг знака = можно, кавычки удобно оставить.)
• После правок обязательно **Ctrl+S** (сохранить).

Напиши агенту одним словом, где застрял: **supabase** / **cursor** / **env**
— и при возможности один скрин (без паролей на экране).
