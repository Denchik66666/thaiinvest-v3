✅ КРАТКО, ЧЁТКО, ПО ОЧЕРЁДНОСТИ — МОИ РЕКОМЕНДАЦИИ (ПЛАН ПРОЕКТА)
Это — профессиональная дорожная карта твоего приложения.
Без лишней воды, только суть.

✅ 1. Dashboard (главная страница)
Почему первым: после логина пользователь должен видеть премиальный, чистый интерфейс.
Что делаем:

создаём контейнер + заголовок
подключаем UI‑KIT
убираем старый разрозненный Tailwind
делаем основу для будущих страниц


✅ 2. Middleware (защита маршрутов)
Почему вторым: чтобы неавторизованный пользователь не попал на Dashboard.
Что делаем:

создаём middleware.ts
проверяем наличие JWT‑cookie
если нет → → редирект /login
если есть → → пропускаем


✅ 3. Navbar / Header
Почему третьим: любое приложение должно иметь верхнюю панель.
Что делаем:

логотип
кнопка выхода
кнопка смены темы (перенесём из layout)
отображение пользователя
премиальная верстка


✅ 4. Sidebar (по желанию)
Почему четвёртым: если у тебя много страниц — нужен современный sidebar.
Что делаем:

список ссылок (Dashboard, Инвесторы, Настройки…)
активный пункт
компактный премиальный дизайн (как Vercel)


✅ 5. Страница Инвесторов (таблица + CRUD)
Почему пятым: это твой основной функционал.
Что делаем:

таблица инвесторов
кнопка "Добавить"
карточка создания / редактирования
интеграция API /investors
UI‑KIT для всех элементов


✅ 6. React Query (если хочешь удобную работу с API)
Почему шестым: ускоряет работу с данными.
Что делаем:

кэш запросов
автоматический refetch
загрузки и ошибки на уровне библиотеки
меньше ручного fetch


✅ 7. Zustand (если хочешь глобальные состояния)
Почему седьмым: удобно для:

данных пользователя
состояния меню
темы (optionally)


✅ 8. API‑рефакторинг (приведение API к идеальному виду)
Почему восьмым: после UI и логики можно навести порядок в API.
Что делаем:

единый формат ошибок
единый формат успеха
защита роутов
валидация входящих данных


✅ 9. Loader / Skeleton / Page transitions
Почему девятым: улучшает UX.
Что делаем:

skeleton загрузка списка инвесторов
анимации
плавные переходы страниц


✅ 10. Профиль пользователя (опционально)
Если нужно:

смена пароля
просмотр информации
роли / права

4️⃣ Investors — таблица + CRUD
