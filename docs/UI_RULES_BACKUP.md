# Резерв правил UI/UX (фиксация договорённостей)

**Назначение:** единая точка правды для агента и команды по визуалу и поведению после финализации кабинета инвестора и шапки.  
**Обновлено:** 2026-05-03  

При расхождении с кодом приоритет у **фактической реализации** в репозитории; этот файл — напоминание «как задумано», чтобы не откатывать случайно.

---

## 1. Дашборд инвестора (`/dashboard`, роль INVESTOR)

- **Единая hero-карточка:** `InvestorPremiumDashboard` — открытая неделя (подпись диапазона), шкала прогресса, прогноз начисления, блок «Доступно к выводу», метрики, встроенная история операций.
- **Шкала недели (градусник):** премиум-**золото только у заливки** (`.thai-investor-thermo-fill`). Трек, «колба», стекло, ambient, тени трубки — **прежняя** палитра (primary / accrued), **не** переводить в золото целиком.
- **Прогноз «Ожидается +… ฿ к …»:** сумма в **премиум-золоте** (класс `.thai-investor-premium-gold-amount` / общая логика с `.thai-investor-forecast-strip__amount`), остальной текст — обычный цвет полосы прогноза.
- **История операций (embedded):**
  - период по умолчанию — **всё время**;
  - список по умолчанию — **полный** (без усечения до первых N строк; кнопка «свернуть / показать ещё» остаётся при длинной ленте);
  - на кнопке выбора периода виден только текст **«Период»** (без второй строки «30 дн.» и т.п.; полный выбор в `aria-label` и внутри попапа);
  - для строки начисления **открытой недели с суммой 0** сумма **`+0 ฿`** — тот же премиум-золотой класс, что и у прогноза (ожидаемое начисление как отдельная «статья»).

---

## 2. Шапка дашборда (sticky bar, все роли)

- В профиль ведёт клик **только по аватару** или **только по нику** (и стрелке у ника); **не** вся растянутая полоса слева от колокольчика.
- **Ник:** класс `.thai-dashboard-nick-matte-gold` — **заметный матовый янтарно-золотой**, без `text-shadow` и без `filter`/свечения; при наведении — лёгкое усиление оттенка, не «неон».

---

## 3. Профиль (`/dashboard/profile`)

- **Смена фото:** загрузка **встроена в круг аватара** (клик по аватару + скрытый `input`); маленькая иконка камеры на круге как подсказка.
- **Отдельной кнопки загрузки на главной дашборда у инвестора нет** — только профиль.
- **Без фото:** **две буквы** инициалов (общая функция `initialsTwoLetters` в `lib/utils.ts`, то же поведение в `UserAvatar`); пустой/пробельный `avatarUrl` считается отсутствием фото.

## 5. Аватары

Реализация: **`POST /api/auth/avatar`** (`app/api/auth/avatar/route.ts`).

| Условие | Значение |
|--------|----------|
| Форматы | **JPEG**, **PNG** (`image/jpeg`, `image/png`) |
| Макс. размер | **2 МБ** |
| Хранение | Файлы **`public/uploads/avatars/{userId}.jpg|png`** (каталог в **`.gitignore`** — в git не коммитить бинарники) |
| Поле в БД | Обновляется **`User.avatarUrl`** |

**Бэкап / прод:**

- Включать каталог загрузок в **резервное копирование сервера** (или вынести в объектное хранилище — тогда отдельная политика версий и URL).
- На Vercel/serverless без постоянного диска — либо внешнее хранилище (S3, Supabase Storage), либо осознанный сброс при редеплое.
